Para personalizar el branding de la app y los reportes, coloca aqui:

1) byrtec_logo.png   <-- Logo en blanco (para usar sobre fondo oscuro)
                          Tamano sugerido: 200x60 pixels o similar
                          Usado en: encabezado del reporte HTML

2) byrtec_icon.ico   <-- Icono de la app Windows
                          Tamano: 256x256 con multiples resoluciones
                          Usado en: dist/Byrtec.exe (al empacar)

3) byrtec_splash.png <-- Imagen del splash screen al iniciar
                          Tamano sugerido: 600x340 pixels
                          Si no existe, se genera uno sintetico

NOTA: el logo blanco que enviaste pegalo aqui como "byrtec_logo.png"
y la proxima vez que generes un reporte HTML aparecera en la cabecera.
